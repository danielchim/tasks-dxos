# tasks-v2

This app was created with a DXOS application template.

- [x] React
- [x] DXOS UI System

Run the app with `npm`:

```bash
npm install
npm run serve
```

Build the app to the `out` folder:

```bash
npm run build
```

Deploy the app to a [DXOS Kube](https://docs.dxos.org/guide/kube):

```bash
npm run deploy
```
